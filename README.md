# Getting started

```bash
git clone git@github.com:loiccth/library-system.git
cd library-system
npm install
cd backend
npm install
```

## Configuration

Open backend folder and rename .env.example to .env  
Configure your settings

## Run dev server

Install development dependencies
Run dev server using Concurrently

```bash
npm run dev
```

## Requirements

1. NodeJS
2. MongoDB Server
3. SMTP Server
4. Twilio API
